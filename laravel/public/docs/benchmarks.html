OPcache / JIT の切替

① その場で切替（PHPBench 流・推奨）

--php-config で計測サブプロセスの ini を上書きする。リビルドも再起動も不要。

# JIT OFF（OPcache は有効のまま）
docker compose exec app vendor/bin/phpbench run benchmarks/MathBench.php --report=aggregate \
  --php-config='opcache.jit_buffer_size: 0'

# OPcache ごと OFF（JIT も連動して無効）
docker compose exec app vendor/bin/phpbench run benchmarks --report=aggregate \
  --php-config='opcache.enable_cli: 0'

② コンテナ全体の既定を変える

環境変数で起動時の ini を生成。Web 側（FPM）にも効かせたい時はこちら。

OPCACHE_ENABLE=0          docker compose up -d   # OPcache 無効
OPCACHE_JIT_BUFFER_SIZE=0 docker compose up -d   # JIT 無効
OPCACHE_JIT=function      docker compose up -d   # JIT を function モードへ
docker compose up -d                             # 既定(tracing)へ戻す

※ 反映には docker compose up -d（コンテナ再作成）を使う。restart は環境変数の変更を拾わない。
